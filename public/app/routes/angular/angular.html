Angular Directives

Click Me!
Click Me!

This illustrates two of Angular's built in directives, ng-hide and ng-show. The first two pictures show how to use both directives. The first div, the one on the left of the screen, has ng-show set to the scope variable 'showLeftBox'. So when that variable is true, the left box will 'show'. First, we add $scope.showLeftBox = true to our Javascript. Then we add ng-show and ng-hide to our divs in our HTML.

The second div has ng-hide set to the same variable. So when that variable is true, div #2 will 'hide'. So one of these boxes will always be hidden, and one will always be showing.

This third picture shows how to switch the variable back and forth. Both divs have an ng-click on them, which changes the variable to the opposite of what it was before the click happened. ng-show, ng-hide, and ng-click are examples of built in directives in Angular. Built in directives make simple animations very easy to implement.
